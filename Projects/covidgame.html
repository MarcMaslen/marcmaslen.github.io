    Covid-19 Java Game

    I made this game during the Covid-19 lockdown for university. The aim of the game was to be informative but also fun. The game is all about social distancing, there are these red sqaures on the map and the longer they touch you the longer your exposed. If your health reaches zero you lose.

    Score is based on time spent alive and there is a leaderboard with the top scores of all time. There is a page which shows you the controls and how to play the game.

    The game did have bugs which I tried to fix, most of them got fixed however, due to time constrant not all of them did. As I was in my second year during lockdown it was even harder to find help and resources. The upside to lockdown was that I had more time to do the project and research making it. The code is availble on my github to look at.
