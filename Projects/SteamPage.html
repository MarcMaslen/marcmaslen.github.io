    Recreating Steam

    This was my attempt at recreating Steam for a project. This was so fun to do and taught me so much as this was one of the first big websites I made. I used HTML, CSS, JavaScript and PHP to create this.

    This assignment gave me a massive grade of 95% and was worth nearly half my module. I created all the pages using HTML, CSS and a bit of Javascript. I then used PHP to create a login system for the page. This was my first time making a login system and it turned out being better then expected. When a user signed up it would be added to a database, then the user could login and stay logged in on each page. They would know there logged in by there username being displayed in the top left corner.
