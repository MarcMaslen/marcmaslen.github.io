    ID Database made in Java with working GUI

    This project is one I did at University, it was a Java application with a working GUI to store six digit ID's. I also had to be able to add, remove, sort and delete them all. Also adding RGB to change the color of the text too.

    This is one of those project that I really enjoyed and was the first time I relised how much I had learnt as I found myself not struggling at all with it. The program seemed to only have bugs when it came to sorting the ID's, other then that it worked flawlessly. I wanted to fix these bugs but the deadline came so needed to submit it. I got a 89% grade for this which I was ecstatic about.

    For some reason our lecturer wanted it all in one class (think it was easier for them to mark) but I commented it clearly so that each section and feature is seperated. I prefer to split my projects up by class as it becomes so much more organised.
